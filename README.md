# CGPA Calculator

A responsive, browser-based CGPA (Cumulative Grade Point Average) calculator that helps students calculate their academic performance.

## Features

- **Dynamic Course Management**: Add or remove courses as needed
- **Grade Point Calculation**: Supports standard 5.0 grading scale
- **Real-time Results**: Instantly calculates CGPA as you input data
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **About Modal**: Learn how CGPA is calculated and its importance
- **Input Validation**: Ensures accurate data entry

## How to Use

1. **Add Courses**: Click "Add Course" to add a new course row
2. **Enter Details**: For each course, provide:
   - Course Name
   - Grade (A-F)
   - Credit Units (1-10)
3. **Calculate**: Click "Calculate CGPA" to see your results
4. **Reset**: Clear all courses and start over with "Reset"
5. **Learn More**: Click "About CGPA" to understand the calculation method

## Grade Scale

| Grade | Points | Percentage | Performance |
|-------|--------|------------|-------------|
| A     | 5.0    | 70-100%    | Excellent   |
| B     | 4.0    | 60-69%     | Very Good   |
| C     | 3.0    | 50-59%     | Good        |
| D     | 2.0    | 45-49%     | Pass        |
| F     | 0.0    | 0-44%      | Fail        |
